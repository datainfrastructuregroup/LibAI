# Example Note

This is an example note that demonstrates the functionality of Obsidian 11ty.

## Features

- [[another-note|Linking to another note]] using double square brackets
- Automatic backlinks
- Graph view of connections

## More Information

You can create links to other notes by surrounding the note name in double square brackets, like this: `[[another-note]]`.

This will automatically create a link to the note with that name, and if the note doesn't exist yet, it will be created when you click the link.
